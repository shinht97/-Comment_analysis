# Review_Analyzer


리뷰를 분석 하여 학습한 모델을 이용 하여 새로운 리뷰의 반응을 파악하고 그 리뷰에 달릴 점수를 예상하는 프로그램

두개의 자연어 처리 모델을 사용하여, 리뷰의 반응을 분석하는 모델과 해당하는 리뷰에 달릴 점수를 예상하는 모델을 사용하여 처리
